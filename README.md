# scriptorium
Literary LaTeX class and WordPress CSS for atmospheric fiction, serialized narratives, and book-like typography.

## Overview

Scriptorium is a small publishing toolkit created to support long-form literary writing across both print-oriented LaTeX workflows and web publication through WordPress.

It focuses on:

- quiet, book-like typography,
- serialized and pericope-based narratives,
- atmospheric fiction,
- consistent visual rhythm between PDF and web formats,
- restrained late nineteenth-century aesthetics.

The project currently contains:

- a LaTeX setup based on the `memoir` class,
- a WordPress CSS stylesheet designed to mirror the printed layout,
- typographic utilities for scene separators, literary notes, quotations, messages, and narrative fragments.

## Philosophy

The goal is not maximal customization or framework complexity.

Scriptorium aims instead for:
- simplicity,
- typographic coherence,
- readability,
- and subtle atmosphere.

Most design choices are intentionally restrained:
historic ligatures, quiet rubrication, moderate spacing, minimal ornamentation, and stable page rhythm.
